❓ 常见问题 ❓ FAQ

🟢 是否需要安装 Python？
不需要。软件已打包为桌面客户端，下载对应系统版本后即可运行。
🟢 Do I need to install Python?
No. The software is packaged as a standalone desktop app — just download and run.
🟢 是否需要蒲公英企业账号？
需要。本工具采集的是蒲公英平台（品牌合作平台）的数据，必须拥有蒲公英企业账号才能登录和使用。个人创作者账号无法使用蒲公英的达人搜索功能。
🟢 Do I need a Pugongying enterprise account?
Yes. This tool scrapes data from the Pugongying brand collaboration platform. You must have an enterprise account to log in. Individual creator accounts cannot access the influencer search features.
🟢 Cookie 怎么配置？需要手动复制吗？
不需要手动复制。软件内置了「Cookie 小工具」，打开后扫码或登录蒲公英平台，工具会自动提取 Cookie，一键完成配置。
🟢 How do I set up the cookie? Do I need to copy it manually?
No manual copying needed. The software includes a built-in Cookie tool — open it, scan QR or log into Pugongying, and it auto-extracts the cookie with one click.
🟢 采集中断后数据会丢失吗？
不会。软件在采集过程中持续保存结果，网络异常或任务中断不会丢失已采集的数据。
🟢 Will I lose data if crawling is interrupted?
No. Results are saved continuously — network issues or interruptions won't wipe collected data.
🟢 能采集哪些数据？有多少个字段？
本工具可采集 34 个核心字段，涵盖博主基本信息、粉丝画像（性别/年龄/地域/兴趣）、商业报价（图文/视频）、合作笔记表现等维度。详见上方「输出数据说明」。
🟢 What data can I collect? How many fields?
The tool collects 34 core fields including influencer basic info, follower profiles (gender/age/region/interests), commercial pricing (image/video), and collaboration post performance. See "Output Fields" above for details.
