核心功能

围绕投放选人流程组织功能：先筛选达人，再整理报价、画像、内容表现和合作笔记。

达人筛选

博主列表采集

按关键词、笔记类型、粉丝量、图文报价和搜索范围等条件筛选达人列表。

报价

商业报价整理

整理图文报价、视频报价、预估阅读单价、合作笔记数和账号评估数据。

画像

粉丝画像采集

采集粉丝数、女性粉丝占比、年龄占比、地域占比等投放参考字段。

笔记

合作笔记表现

整理合作笔记的阅读数、互动数和阅读来源占比，用于复盘达人质量。

使用流程

软件不替代蒲公英权限。开始前先确认账号可以正常访问蒲公英页面和相关数据。

确认账号权限使用具备蒲公英访问权限的企业资质账号。
下载客户端从 GitHub Releases 下载最新版 Windows 或 macOS 客户端。
配置 Cookie使用软件内置 Cookie 小工具完成登录态配置。
设置筛选条件填写关键词、笔记类型、粉丝量、报价区间和搜索范围等条件。
查看结果采集完成后，在软件所在目录查看 CSV 文件和运行日志。

常见问题

这里回答购买前最常见的问题。更细的字段说明、技术说明和反馈格式请查看 GitHub README。

没有蒲公英权限可以用吗？

不可以。该工具依赖蒲公英页面和权限数据，没有权限的账号无法正常访问相关数据。

是否需要安装 Python？

不需要。软件已打包为桌面客户端，下载对应系统版本后即可运行。

为什么不同账号采集结果不同？

蒲公英数据可能与账号权限、企业资质、平台展示策略有关，不同账号看到的数据可能不同。

采集中断后数据会丢失吗？

软件会在采集过程中持续保存结果，已完成部分通常会保留在 CSV 文件中。

准备整理蒲公英达人和投放数据？

先确认账号权限，再下载最新版或进入商品页开通授权。
